# 📡 Telecom X - Inteligencia Predictiva (Parte 2)

Este proyecto aplica técnicas avanzadas de **Machine Learning** para predecir la evasión de clientes. Se centra en la transformación de datos para modelado, balanceo de clases y evaluación de algoritmos de clasificación.

## 🚀 Hitos del Proyecto
- **Encoding Multivariante:** Transformación de 21 variables categóricas a formato numérico.
- **Balanceo SMOTE:** Mitigación del sesgo de clase para mejorar la detección de fugas reales.
- **Optimización de Modelos:** Comparación de desempeño entre Logistic Regression y Gradient Boosting.
- **Interpretación:** Análisis de importancia de variables para toma de decisiones de negocio.

## 📊 Resultados Técnicos
- **Modelo:** Gradient Boosting Classifier.
- **Recall:** ~82% (Detección efectiva de clientes en riesgo).
- **KPI Clave:** Se identificó que el **Tipo de Contrato** y los **Cargos Mensuales** explican el 65% de la probabilidad de abandono.

## 🛠️ Requisitos
- Python 3.8+
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Pandas / Seaborn
